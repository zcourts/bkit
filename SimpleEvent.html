Mixin: SimpleEvent

SimpleEvent

Container for an event that can be dispatched. Effectively, wraps js-signal's "Signal" type. It borrows ideas from the built in Event object for more info {@see https://developer.mozilla.org/en/docs/Web/API/Event}
